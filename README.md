# QR Code Generator

A feature-rich QR Code Generator built using HTML, CSS, and JavaScript. This application allows users to generate, customize, download, and manage QR codes with advanced functionality such as analytics, URL shortening, batch generation, Bio Link QR creation, and social sharing.

## ✨ Features

### QR Code Generation
- Generate QR codes for URLs, text, email addresses, phone numbers, and Wi-Fi credentials.
- Adjustable QR code size and error correction levels.
- Custom foreground and background colors.
- Logo/Image embedding support.

### Download & Export
- Download QR codes as PNG, JPEG, and WebP.
- Print QR codes directly.
- Copy QR image data to clipboard.
- Share QR codes using the Web Share API.

### History Management
- Save previously generated QR codes.
- Delete individual history entries.
- Clear complete history.
- Local storage persistence.

### URL Shortener
- Generate shortened URLs.
- Auto-generate QR codes for shortened links.
- Copy shortened URLs instantly.

### Batch QR Generation
- Generate multiple QR codes simultaneously.
- Supports up to 20 entries per batch.
- Fast processing with visual previews.

### Analytics Dashboard
- Track total QR codes generated.
- Daily generation statistics.
- Export tracking and usage analytics.
- Interactive charts powered by Chart.js.

### Bio Link QR Generator
- Create multi-link QR profiles.
- Add and manage multiple links.
- Generate a single QR code containing all profile links.

### Social Sharing
- Share QR codes on:
  - Facebook
  - Twitter/X
  - LinkedIn
  - WhatsApp
  - Telegram
  - Pinterest

### User Experience
- Fully responsive design.
- Dark and Light mode support.
- Real-time feedback messages.
- Loading indicators.
- Keyboard shortcut support.
